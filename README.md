# Rikki

## 项目简介
Rikki 是一个基于 Next.js 15 和 React 19 构建的现代化 Web 应用程序，采用了最新的前端技术栈和工具链。项目使用 TypeScript 进行开发，确保了代码的类型安全性和可维护性。

## 部署方式

### 环境要求
- Node.js (推荐最新 LTS 版本)
- pnpm 10.5.2 或更高版本

### 安装步骤
1. 克隆项目代码
```bash
git clone https://github.com/ave-mygo/rikki.top.git
cd rikki.top
```

2. 安装依赖
```bash
pnpm install
```

3. 开发环境运行
```bash
pnpm dev
```

4. 生产环境构建
```bash
pnpm build
pnpm start
```
